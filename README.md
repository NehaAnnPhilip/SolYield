SolYield - FIELD TECHNICIAN ASSIST:

SolYield is a React Native mobile application designed to assist field technicians 
in managing remote solar park maintenance. 

LEVEL 1 FEATURES:
1. Schedule: A list of visits that syncs to the phone calendar. (You are ~50% done here).
2. Check-In: A button that uses GPS to say "Success" or "Too far away."
3. Map: A map view showing pins for the solar farms.
4. Report: A button that creates a simple PDF summary of his work.

TECH STACK:
Framework: React Native (Expo)
Language: TypeScript
Navigation: React Navigation (Bottom Tabs)
Libraries: expo-calendar, expo-location, expo-print, react-native-maps
